# Перечисления

Важно! Всё описанное в данной статье может не попасть в язык, это больше концепт, чем документация.

В основном язык рассчитан на создание переводов, но иногда бывают случаи, когда вам нужно сохранить значения, которые будут считаться просто значениями и будут для всех локализаций одинаковы.

В таком случае вы можете поместить все желаемые значения в перечисление.

Перечисления могут иметь как просто значения, так и функции или новые перечисления.

Важно:

* Перечисление не может быть пустым

* Перечисление желательно называть с заглавной буквы E

* Перечисления не могут переопределять уже определённые значения

* Перечисление с одним и тем же именем можно объявлять много раз, тогда все значения будут учитываться вместе

## Объявление

Чтобы объявить перечисление вам нужно написать enum и его название. 



Создадим перечисление:

```ling
enum EHeaderTextColor {
    AQUA = "#00ffff"
    ORANGE = "#ff6600"

    getColor(text) {
        EHeaderTextColor.random("value")    
    }
}
```

В случае, если вы находитесь внутри пакета и вам хочется, чтобы данные помещались внутри него, вы можете описать поле enum без имени:

```ling
package example {
    enum {
        text = "This is example"
    }
    test = example.text //This is example
}
```

Или просто сделайте универсальный пакет, но тогда все переводы с одним значением будут по-умолчанию считаться обычным значением из перечисления:

```ling
define langs = en-US, ru-RU
package enum example {
    text = "This is example" //это значение
    translation = "Text: ${text}", "Текст: ${text}" //это перевод
}
```

## Методы перечислений

Перечисления сами по себе не хранят методы, но при их использовании в коде их вызывают.

#### Случайное значение

Для того, чтобы получить случайное значение из перечисления, вы можете использовать метод random:

```ling
enum ERandomNumber {
    one = 1
    two = 2
    three = 3
    rand_value = ERandomNumber.random("value") //будет работать только со значениями
    rand_function = ERandomNumber.random("function") //будет работать только с функциями. Если их не будет, вернёт null и выдаст предупреждение
}
```

#### Порядковый номер

Вы можете получить порядковый номер элемента при помощи метода ordinal. Элементы, как и в массиве, начинаются с нулевого индекса:

```linga
enum ERandomNumber {
    one = 1
    two = 2
    three = 3
    ordinal_two = ERandomNumber.ordinal(two) //1
}
```
