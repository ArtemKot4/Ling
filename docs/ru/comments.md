# Комментарии ☑
Комментарии позволяют писать заметки в коде и не учитываются.

## Однострочные
Объявляются при помощи `//`

```ling
//однострочный комментарий
``` 

## Многострочные
Открываются при помощи `/*`, закрываются при помощи `*/`

```ling
/*
Я многострочный комментарий.
Могу быть в неограниченное количество строк.
*/
```

## Документация
Это комментарии, которые вы можете увидеть при наведении на функцию, пакет или перевод.

Открываются при помощи `/**`, закрываются при помощи `*/`

```
/** 
 * Я показываюсь при наведении на example
 */

package example {
    /** 
     *Я показываюсь при наведении на message
     */
    message() {
        "пример"
    }
}
```
