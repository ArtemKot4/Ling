# Определения ☑:

Определения - важная особенность языка Ling. Она позволяет создавать переводы сразу под несколько языков или создавать метод, который будет возвращать указанное вами выражение в случае, если запрошенного перевода не существует. Все параметры необязательны.

## Параметры
Здесь вы ознакомитесь со всеми допустимыми параметрами. В случае попытки указать другие, вы получите ошибку.
#### Параметр указания языков: lang
Данный параметр позволяет перечислить через запятую языки, которые мы сможем использовать на наших функциях и в наших переводах.

Вы не можете повторять переводы в одном перечислении, а так же следующее определение, созданное в этом же пакете, перезапишет их.

В случае, если перевода не окажется в перечислении, мы получим соответствующую ошибку.

Пример использования:
```ling
define langs = en-US, ru-RU
```
#### Параметр кодировки файла: encoding
В данный момент не применяется, но может быть указан.
Отвечает за кодировку при чтении файла с расширением ling.

Пример использования:
```ling
define encoding = "utf-8"
```

#### Метод для обработки несуществующих переводов: unexpected
Данный метод позволяет обрабатывать ситуации, в которых запрашивается перевод по ключу, которого не существует. 

Таким образом вы можете описать собственное поведение того, какой результат вернётся вместо стандартного.

В случае, если пакета или перевода не существует, будет искаться функция unexpected по всей иерархии, и если её не окажется, вернётся стандартное "Unknown key: название". 

Пример использования:
```ling
define unexpected(key) {
    "Вам повезло, даже тестеры не наткнулись на отсутствие меня: ${"key"}, а вы смогли!"
}
```

## Краткая форма записи
Вы можете определить параметры в краткой форме. Для этого достаточно написать ключевое слово define в любой части кода и название ключа со значением.
```ling
//для параметров
define название = значение 

//для функций
define название(аргументы) {
    выражение
} 
```
## Запись в виде блока
Вы можете определить сразу все необходимые вам параметры, если после define добавите объявление блока:
```ling
define {
    langs = en-US, ru-RU //переводы
    encoding = "utf-8" //кодировка
    unexpected(key) {
        key
    } //метод
}
```
