# Ling

#### Как пакетный менеджер

Ling - пакетный менеджер, предоставляющий возможности написания переводов на языке проектирования одноимённого формата.

#### Как язык проектирования

Язык ling представляет собой DSL, написанный для более удобного создания переводов засчёт собственного синтаксиса и системы пакетов.

В пакеты можно запаковывать переводы и функции их обработки.

Файлы могут поддерживать многоязычность, если перечислены в определениях. В таком случае переводы пишутся через запятую. 

##### Планируется:

Поддерживаются комментарии и различные операторы, позволяющие вынести логику обработки переводов прямиком внутрь них.

Язык на выходе превращается в json, которые использует пакетный менеджер. Язык транспилируется только в том случае, если файл изменился.
