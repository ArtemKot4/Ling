# Ling Format

Тут описана разметка, которая должна быть допустима в любом ling файле.

## Определения:

Определения позволяют сделать возможной поддержку некоторых настроек. Могут быть заданы в любой строке файла. Все настройки являются опциональными.

```ling
define {
    langs = en-US //языки в формате lowercase-UPPERCASE через запятую
    encoding = "utf-8"
    unexpected(key) {
        "Ключ " + key + " не найден!"
    }
}
```

Или более сокращённая форма под конкретный параметр:

```ling
//define название = значение

define langs = en-US
define encoding = "utf-8"
define unexpected(key) {
    "так тоже можно"
}
```

Определений может быть неограниченное количество. Языки из определений складываются, в то время как другие параметры перезаписываются. Языки не должны повторяться.

## Пакеты:

Пакеты позволяют запаковывать переводы в нечто похожее на объекты json. Создаются при помощи ключевых слов `package`.

```ling
define langs = ru-RU, en-US

package navbar {
    package faq {
        about_us = "О нас", "About us"
    }
}

//или 

package navbar.faq {
    idea = "Идея", "Idea"
}


//перевод, выдаваемый в случае попытки получить незарегистрированный перевод в пакете
package navbar.faq {
    unexpected(key) {
        "Ключ " + key + " не найден в navbar.faq"
    }
}
```

## Переводы:

Переводы помещаются в пакеты и могут быть заданы в одну строку или в несколько.

#### В одну строку:

```ling
define ru-RU, en-US

package navbar.settings {
    theme = "Тема", "Theme"
}
```

#### В несколько строк:

```ling
define ru-RU, en-US

package navbar.settings {
    color {
        "цвет",
        "color"
    }
}
```

Причём строка не имеет ограничений по переносам строк и может быть записана например так:

```ling
define rome-LAT

package example {
    forest = "
        Infinite Forest 
        est silva in lux,
        non locus in malum.
    "
}
```

## Комментарии:

Комментарии позволяют делать заметки в коде и ни на что не влияют.

#### Однострочные:

```ling
//комментарий
```

Многострочные:

```ling
/* комментарий */
```
